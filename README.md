# EDA_Cricket_Matches
IPL 2022 - Exploratory Data Analysis
A EDA project on the IPL 2022 season using Python. The dataset contains 74 matches and 20 columns covering match results, player performances, toss decisions, and venue details.

Analysis Performed: 

Which team won the most matches?
Toss decision trends and toss vs match winner correlation.
Top 10 Player of the Match award winners.
Top run-scorers and best bowling figures.
Most matches played by venue.
Highest winning margin, highest individual score, and best bowling figure.

Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn
